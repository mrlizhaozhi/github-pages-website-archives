INSIGHTS

This is a minimalist website. As I need to prioritise other, more important assignments in precious free time, I focus most effort on maintaining the content on this website instead of its design. However, I will certainly improve the layout when I have time to do so. Please bear with me in this transition.

In fact, I didn’t write the code behind the pages at all. Just right click anywhere and select “View Page Source”, and a new tab would open with the source code of the page. A short glance should reveal that this page is converted from a word document. In fact, Warren Buffett’s multi-billion dollar investment firm Berkshire Hathaway Inc.’s official website is built in a similar way.
